# Netlify Apps

This folder contains the shell and scaffolding used to create Netlify app projects only.

## Purpose

- Hosts the base structure for Netlify apps
- Is not intended for application business logic or data processing
- Serves as a container for app manifests, deployment config, and lightweight app wrappers

## Contents

- `mztradingdata/` — Netlify app source and configuration
- `netlify.toml` — Netlify deployment settings for the app shell

## Notes

Keep this folder focused on Netlify app setup and hosting-related assets. Actual app implementation and data generation belong elsewhere in the repository.
